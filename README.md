# Judah Abraham — Portfolio

## File Structure
```
judah-abraham-portfolio/
├── index.html          ← Main portfolio page
├── project.html        ← Project detail page
├── style.css           ← All styles
├── script.js           ← All JavaScript
├── data/
│   └── projects.js     ← Edit this to update your projects
└── assets/
    ├── judah-abraham-cv.pdf   ← Add your CV here
    └── images/
        ├── pixvault-1.jpg     ← Add real screenshots
        ├── pixvault-2.jpg
        ├── pixvault-3.jpg
        ├── kaad-1.jpg ... etc
        └── (named: projectid-1.jpg, projectid-2.jpg, projectid-3.jpg)
```

## To customise
- Edit `data/projects.js` to change project details, URLs, technologies
- Drop screenshots into `assets/images/` — they will auto-load
- Add your CV as `assets/judah-abraham-cv.pdf`
- Replace social links in `index.html` (search for instagram.com, github.com etc)
- To connect the contact form, update `handleFormSubmit` in `script.js`
